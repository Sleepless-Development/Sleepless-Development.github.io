# disableInteract

Disables or enables the interaction system and clears nearby/current options.

### Parameters
- `state` (`boolean`): `true` to disable, `false` to enable

### Example
```lua
-- Using interact namespace
interact.disableInteract(true)  -- Disable interactions
interact.disableInteract(false) -- Enable interactions

-- Using export
exports.sleepless_interact:disableInteract(true)  -- Disable interactions
exports.sleepless_interact:disableInteract(false) -- Enable interactions
```
