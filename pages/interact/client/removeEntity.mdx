# removeEntity

Removes an existing interaction associated with a networked entity by its network ID. This function facilitates the management of dynamic interactions within the game world, allowing for the removal of interactions tied to networked entities that have been despawned or are no longer relevant.

## Parameters

- `netId` (`number`): The network ID of the entity whose interaction you wish to remove.

## Example

```lua
-- To remove the interaction associated with this networked vehicle:
interact.removeEntity(networkId)
